## Short Replies

### What is live?

The public demo runs against a controlled web app so the test flow stays stable and easy to reproduce.

### Why not test a random public site?

We wanted a repeatable demo target with deterministic copy, routes, and layout so the launch reflects the product instead of a third-party UI change.

### What should I try first?

Describe a login or shopping flow in natural language and run it against the demo storefront.
