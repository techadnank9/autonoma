# Launch Checklist

## Product

- Deploy `apps/demo-target` to `demo.autonoma.app`
- Deploy `apps/ui` to Vercel
- Deploy `apps/api` to Render
- Apply production migrations
- Seed the demo organization/application data

## Demo

- Verify `Demo App` resolves to `https://demo.autonoma.app`
- Run a login happy path
- Run a search/products happy path
- Run an add-to-cart happy path
- Confirm result output is clean enough for recording

## Media

- Capture one short demo video
- Capture one screenshot or GIF
- Verify mobile readability of the media

## Social

- Paste the demo URL into the primary launch post
- Prepare two follow-up replies
- Validate that the public links work before posting
